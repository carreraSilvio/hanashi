# Hanashi Narrative Editor
A node-based Narrative Editor for Unity
![hanashi-001](https://user-images.githubusercontent.com/3154359/126072143-9f44e023-feaa-48ea-90d8-28f9a8797852.gif)

# Features
* Text Node: Node with a speaker and message
* Choice Node: Node with an option to set up choices to branch the narrative 

# Roadmap
* Local Variables: create variables to set and check values and use on your narrative
* Global Variables: set variables and use them across different narratives
* Character Database: create characters to easily set who's the speaker
* Message Effects: Set the color of the text, size, delay between words.
